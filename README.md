# Anushka_MLT_Lab_CM23028

Machine Learning Techniques Lab Practicals

This repository contains implementations of various **Machine Learning Techniques (MLT) lab experiments** performed as part of the academic coursework. The experiments demonstrate fundamental machine learning concepts including regression, image classification, digit recognition, sentiment analysis, and real-time webcam based applications.

---

## 📚 Practicals Included

### Practical 01 – Linear Regression

Implementation of **Linear Regression** for predicting values based on input data.

**Concepts Covered**

* Supervised Learning
* Regression Analysis
* Model Training and Prediction

---

### Practical 02 – Digit Recognition

Digit classification using the **MNIST dataset** and a **Convolutional Neural Network (CNN)**.

**Concepts Covered**

* Image Processing
* Deep Learning
* CNN Architecture
* Model Evaluation

---

### Practical 03 – Text Sentiment Analysis

A Natural Language Processing project to classify text sentiment (positive/negative).

**Concepts Covered**

* Text Preprocessing
* Feature Extraction
* Sentiment Classification

---

### Practical 04 – Image Classification

Implementation of image classification using machine learning / deep learning models.

**Concepts Covered**

* Image Dataset Processing
* Model Training
* Prediction and Accuracy Evaluation

---

### Practical 05 – Webcam Based Object / Gesture Detection

A real-time application using webcam input for classification or detection.

**Concepts Covered**

* Real-Time Image Capture
* Computer Vision
* Model Prediction

---

### Prelab – Hello TensorFlowJS

Introduction to **TensorFlow** and **TensorFlow.js**.

**Concepts Covered**

* Running ML models in the browser
* Basic TensorFlowJS setup
